Projeto de aplicativo Android que calcula o índice de massa corporal, o IMC. Um aplicativo funcional e com ótima usabilidade. Na primeira tela o usuário digita os dados de peso e altura,
e clica no botão de calcular, então o aplicativo passará para a próxima tela, onde mostrará os dados digitados e o diagnóstico de acordo com os parâmetros do IMC.

Aplicativo totalmente desenvolvido no Android Studio utilizando linguagem Kotlin e layout em xml.

Tecnologias utilizadas:
- Activities
- Bundle
- Intent
- ConstraintLayout
- ImageView
- TextInputLayout
- Button
- TextView
